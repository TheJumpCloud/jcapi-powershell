# Get-JcSdkCommand

## SYNOPSIS
This endpoint returns a specific command based on the command ID.

#### Sample Request
```
curl -X GET https://console.jumpcloud.com/api/commands/{CommandID} \\
  -H 'Accept: application/json' \\
  -H 'Content-Type: application/json' \\
  -H 'x-api-key: {API_KEY}'
```
